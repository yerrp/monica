AN APP FOR MANUAL
RUN TIMING
LEARN HOW CLOSE GET STARTED
  • ADD A CSV FILE
    Use a Comma Separated File with Bib Number, First Name, Last Name, Age, & Sex. Create a CSV file using Google Drive or Use Sample
  • START THE TIMER
    The on screen clock will begin.
  • RECORD A TIME
    Click the record button. Click the record time box. Press Enter when a runner crosses the finish line.
  • RECORD A RUNNER
    Click the record button. Click the record runner box. Type in the bib number. Press Enter. Made an error? Double click a runner to undo a recording or click & drag to manually sort finished runners.
  • PRINT THE RESULTS
    Click the finished button. Type in a race name & race date. Print!
  • IT'S THAT EASY
    Remember. Timing is serious business. Practice & always have a plan b.
